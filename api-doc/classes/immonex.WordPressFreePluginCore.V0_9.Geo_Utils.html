geocode()

geocode(string  $address, string  $return_type = 'compact', array<mixed,string>|boolean  $use_providers = false, array  $keys = array(), string  $language = 'de', string|boolean  $countrycodes = false) : object|array|boolean

Get geo information for a given address.

Parameters

string $address

Address for geocoding.
string $return_type

"full" or "compact" (the latter one only returns coordinates).
array<mixed,string>|boolean $use_providers

Geocoding API to use in given order until a valid result is returned (false = use default provider list).
array $keys

Array of API keys (if required).
string $language

ISO-2 Language code for results (if supported, optional, default "de").
string|boolean $countrycodes

Comma separated list of ISO-2 country codes for limiting the geo query ("germany_and_neighbors", "eu" or "europe" for respective default lists), false for no country limiting.

Returns

object|array|boolean —

Geo information/coordinates, false if geocoding failed.
